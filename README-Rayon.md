Require
-------
- OpenCL device


Additional Options
------------------
--no-gpu           Do not use GPU, even if GPU is available.

--no-early-pass    Do not pass.
                   (for CGOS)

--device-id 0      Set GPU to use.
                   --device-id X where X is an integer >=0 means use GPU X, i.e. deviceId=0 means GPU 0, etc.
